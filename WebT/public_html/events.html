Fehler:

Event Manager

Der Evenet Manager ist für Google Chrome (Version 28.0.1500.52) ausgelegt. Wenn Sie noch kein Login haben einfach Benutzername und Passwort eingeben und anschließend auf "Registrieren" klicken. Über "Event hinzufügen" können Sie neue Events hinzugefügen, an denen andere Benutzer teilnehmen können. Um selber an einem Event teilnehmen zu können, einfach die Lupe des gewünschten Events anklicken und im Dialogfenster auf "Teilnehmen" klicken. Über "Logout" können Sie sich von der Anwendung abmelden. Alle im Browser gespeicherten Daten werden dann gelöscht.

Viel Spaß!
